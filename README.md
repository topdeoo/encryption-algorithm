# encryption-algorithm

## MD5

使用 `C++` 实现 `MD5` 算法。
在 `Ubuntu20.04.4` 上通过测试。
具体而言：
- 可以通过运行 `make` 进行构建
- 通过命令 `./main -md5` 运行程序
- 在命令行中输入字符即可转化为 `md5` 码，输入 `q` 以退出

## RSA

使用 `C++` 实现的简单 `RSA` 算法
在 `Ubuntu20.04.4` 上初步通过测试。
暂时只能够加密数字与字符，字符串等暂未实现。
具体而言：
- 可以通过运行 `make` 进行构建
- 通过命令 `./main -rsa` 运行程序
- 在命令行中输入字符，可得到加密后的字符串，并通过解密算法得到明文。输入 `-1` 以退出。
